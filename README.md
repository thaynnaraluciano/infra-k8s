# infra-k8s

Repositório para armazenamento da infraestrutura compartilhada entre os microsserviços de Donation e Campaing do hackathon da turma 9NETT da Pós Tech da FIAP. 

*namespace* -> diretório que contém o arquivo namespace.yaml responsável por criar o namespace em que a aplicação irá executar.

*k8s* -> diretório que contém os arquivos yaml de toda a infraestrutura compartilhada: MySQL, RabbitMQ, Prometheus e Grafana.
